`git rebase commitid`

该命令可以把另一个分支的最新 commit 改变为当前分支的基础。在改动中会把不一致的本分支 commit 保存，把另一个分支的 commit 加载过来，然后把之前保存的 commit 打上新的时间戳放在后面，简称“变基”。**rebase 过程是与提交时间无关的**。
